# GAPP Standalone Demo

## Quick Start

1. **Install Node.js** (if not already installed)
   - Download from: https://nodejs.org/
   - Version 16 or higher required

2. **Install dependencies:**
   ```bash
   cd frontend
   npm install
   ```

3. **Start the demo:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   ```
   http://localhost:3000/demo
   ```

5. **Click "Yes, I am Mr. Ahmed"** to start exploring!

## Troubleshooting

- **Port 3000 in use?** Change port in `frontend/vite.config.ts`
- **npm install fails?** Make sure Node.js v16+ is installed
- **Demo not loading?** Check browser console (F12) for errors
